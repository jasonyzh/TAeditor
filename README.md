# TA Editor

基于 Vue 3 的所见即所得结构化文档编辑器，专为编写"异常体调查报告"设计。

## ✨ 特性

### 🎨 现代化界面

- **暗色主题设计**：专业的蓝色调界面，专注编辑体验
- **A4 纸张实时预览**：所见即所得，精确控制页面布局
- **响应式布局**：左侧组件面板 + 右侧编辑区域

### 📝 结构化编辑

- **层级标题系统**：H1/H2/H3 标题，清晰的内容结构
- **丰富的区块类型**：
  - 📋 **异常体档案**：支持历史、焦点、领域等多个维度
  - 🔍 **调查前夕**：晨会记录、任务简报、可选目标
  - 🌀 **混沌效应**：表格化混沌现象记录
  - 📖 **调查**：多级调查流程记录
  - 📊 **余波**：结果追踪与申领物管理
  - 📄 **自由区块**：自定义内容区块

### 🧩 组件系统

- **申领物组件**：记录获得的物品和数量
- **异常能力组件**：记录特殊技能和触发条件
- **文本区块组件**：支持背景色的彩色文本区域
- **可选目标系统**：嘉奖⭐/申诫⚠标记，支持切换

### 💾 模组管理

- **多模组支持**：同时管理多个调查报告
- **完整的CRUD操作**：创建/读取/更新/删除模组
- **模组重命名**：直观的内联编辑功能
- **智能数据持久化**：自动保存至 localStorage，500ms 防抖
- **导入导出**：JSON 格式备份和恢复

### 📤 多格式导出

- **PDF 导出**：使用浏览器原生打印，完美分页
- **Word 导出**：
  - 支持 .docx 格式
  - 异常体档案左右分栏布局
  - 混沌效应表格化呈现
  - 可选目标图标标识
- **图片导出**：高清 PNG 截图，2x 分辨率
- **JSON 导出**：完整数据结构，便于备份和迁移
- **统一文件命名**：`[模组名]_[时间戳].[扩展名]`

### 🎯 用户体验

- **自动保存**：实时保存，永不丢失数据
- **保存状态指示**：保存中... → 已保存 ✓
- **插入标记功能**：在文本中插入特殊图标（普通/特殊）
- **颜色文本**：多种背景色选择，突出重要内容
- **内联SVG图标**：无需外部资源，加载更快

## 🛠️ 技术栈

- **Vue 3** + **TypeScript** + **Vite**：现代化前端框架
- **Pinia**：轻量级状态管理
- **docx**：专业 Word 文档生成
- **html2canvas**：高清截图导出
- **file-saver**：文件下载处理
- **组件化架构**：高度模块化的代码结构

## 📦 安装

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd TAeditor

# 安装依赖
npm install
```

## 🚀 开发

```bash
# 启动开发服务器
npm run dev

# 应用将在 http://localhost:5173 启动
```

## 🏗️ 构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

构建产物为纯静态文件，可直接部署到任何静态服务器。

## 🎮 使用指南

### 基本操作

1. **选择模组**：点击顶部工具栏的模组选择器
2. **编辑内容**：直接在右侧编辑区域输入内容
3. **插入组件**：
   - 点击目标板块使其激活
   - 在左侧组件面板选择组件类型
   - 点击"插入"按钮
4. **导出文档**：选择需要的导出格式，文件自动命名下载

### 高级功能

- **插入标记**：在富文本编辑器中插入特殊图标
- **文本区块**：在任意位置插入彩色背景的文本区域
- **可选目标**：添加可选目标，点击图标切换嘉奖/申诫状态
- **模组管理**：创建新模组、重命名、删除等操作

### 快捷技巧

- **自动保存**：编辑后自动保存，无需手动操作
- **内联编辑**：模组名称点击编辑图标即可重命名
- **批量操作**：支持同时操作多个组件
- **样式自定义**：文本区块支持多种背景色

## 🔧 配置与定制

### 样式定制

项目使用CSS变量定义主题颜色，可在 `src/assets/` 中修改：

```css
:root {
  --c-tech-blue: #0066FF;
  --c-alert-red: #FF2D55;
  /* 更多颜色变量... */
}
```

### 导出设置

导出格式和样式可在 `src/utils/` 中对应文件修改：

- `export-word.ts`：Word 导出格式
- `export-pdf.ts`：PDF 导出设置
- `export-image.ts`：图片导出参数

## 🐛 故障排除

### 常见问题

1. **数据丢失**：检查浏览器 localStorage 是否启用
2. **导出失败**：确认有足够的浏览器权限
3. **样式异常**：清除浏览器缓存后重试

### 浏览器兼容性

- 推荐使用现代浏览器（Chrome、Firefox、Edge最新版本）
- 不支持 IE 浏览器
